# 🚀 Danial Hirzan Akbary - Personal Portfolio

## 🎨 Tampilan Section/Fitur

### 1. **Navbar (Navigasi)**
- **Tampilan:** Navigation bar fixed di atas dengan background gelap, logo "DANIAL" berwarna biru, dan menu navigasi (Home, About, Certificates)
- **Fitur:**
  - Sticky positioning (tetap di atas saat scroll)
  - Responsive mobile menu dengan hamburger icon
  - Hover effect pada link navigasi
  - Smooth scroll ke section terkait

### 2. **Hero Section (Beranda)**
- **Tampilan:** Layout 2 kolom (teks kiri + foto kanan) dengan background gelap
- **Fitur:**
  - Nama besar dengan highlight warna biru
  - Deskripsi peran (Mahasiswa Sistem Informasi)
  - Bio singkat dengan gaya humor
  - 2 tombol CTA (Call-to-Action): "Lihat Profil" dan "Sertifikat"
  - Foto profil berbentuk rounded square dengan border

### 3. **About Section (Tentang Saya)**
- **Tampilan:** Section dengan judul, deskripsi, dan grid hobbies
- **Fitur:**
  - **Hobbies List:** 8 item hobby dalam grid 2 kolom dengan icon Font Awesome
  - **Skills Grid:** 6 skill card dalam grid 3 kolom, masing-masing dengan:
    - Icon skill
    - Nama skill
    - Progress bar dengan animasi width
    - Persentase skill

### 4. **Certificates Section (Sertifikat)**
- **Tampilan:** Grid 3 kolom berisi card sertifikat
- **Fitur:**
  - Card dengan icon badge di header
  - Informasi: Judul sertifikat, penerbit (INFORSA), tahun
  - Tombol "Lihat Sertifikat" yang mengarah ke file PDF
  - Hover effect dengan shadow dan border highlight
